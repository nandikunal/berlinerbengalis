```html
<!-- Keywords (6-8 primary + secondary) -->
<meta name="keywords" content="...">

<!-- Open Graph (Facebook Sharing) -->
<meta property="og:type" content="website">
<meta property="og:image" content="...1200x630px...">

<!-- Twitter Card -->
<meta property="twitter:card" content="summary_large_image">

<!-- Canonical URL (prevents duplicate content) -->
<link rel="canonical" href="https://berlinerbengalis.com/...">

<!-- Schema.org JSON-LD -->
<script type="application/ld+json">{ Organization/Event/etc }</script>
```

## üöÄ Content Optimization Tips

### Title Tags Best Practices
- **Length**: 50-60 characters
- **Format**: Primary Keyword | Brand Name
- **Example**: "About Berliner Bengalis e.V. | Bengali Culture Berlin"
- **Include**: Primary keyword at the start
- **Avoid**: Keyword stuffing, truncation on mobile

### Meta Descriptions Best Practices
- **Length**: 150-160 characters (full display)
- **Format**: Action-oriented + benefit
- **Include**: Primary keyword naturally
- **Call-to-action**: "Learn more", "Discover", "Join us"
- **Avoid**: Keyword stuffing, duplicate descriptions

### Heading Structure Best Practices
- **One H1**: Main page topic
- **Multiple H2s**: Major sections (Events, Gallery, About)
- **H3s**: Subsections under H2s
- **Don't skip levels**: H1 ‚Üí H2 ‚Üí H3 (not H1 ‚Üí H3)
- **Include keywords**: Naturally in headings

### Content Best Practices
- **Length**: 300+ words per page (aim for 1000+)
- **Keywords**: 1-2% keyword density (natural)
- **LSI Keywords**: Related terms (Bengali, culture, festival)
- **Internal Links**: 2-3 per 1000 words
- **Images**: Optimized with alt text

## üé® Image Optimization Best Practices

### For Current & Future Images

```html
<!-- Proper image markup -->
<figure>
  <img src="/images/pandal.jpg" 
       alt="Decorated Durga Puja pandal with traditional Bengali architecture"
       width="800" 
       height="600"
       loading="lazy">
  <figcaption>Beautiful pandal decoration showcasing Bengali artistic traditions</figcaption>
</figure>
```

### Alt Text Guidelines
- **Be descriptive**: Describe what's in the image
- **Include keywords**: Naturally if relevant
- **Short but complete**: 100-125 characters
- **Not keyword stuffing**: "Durga Puja, Bengali culture, Berlin, celebration"
- **Function description**: For icons/logos

### Image Formats
- **JPG**: For photographs (best compression)
- **PNG**: For graphics with transparency
- **WebP**: Modern format (smaller file size)
- **SVG**: For logos and icons

### Optimization Tools
- TinyPNG.com: Compress images online
- ImageOptim: Desktop compression
- Cloudinary: Cloud image optimization

## üîó Internal Linking Strategy

### Link Placement
- Navigation menu links to main pages
- Hero CTA buttons to event/gallery pages
- Footer links to all main pages
- Content links within text (natural, 2-3 per page)

### Anchor Text Guidelines
- **Use descriptive text**: "Learn more about Durga Puja" vs "Click here"
- **Include keywords**: Naturally relevant to target page
- **Avoid generic**: Not "Link", "Page", "Article"
- **Variety**: Different anchor text for same page

### Internal Link Examples
```html
<!-- Good -->
<a href="/pages/event.html">Explore Durga Puja 2025 event schedule</a>

<!-- Better (includes keyword) -->
<a href="/pages/gallery.html">View our Durga Puja celebration gallery</a>

<!-- Bad -->
<a href="/pages/about.html">Click here for more info</a>
```
